Moving Average Crossover Strategy

Quantitative trading system based on systematic signal generation and backtesting.

Overview

This project implements a quantitative trading strategy using moving average crossovers applied to financial time series data. It focuses on systematic rule-based decision making and evaluates performance through structured backtesting.

Key Features

• Market data retrieval using yfinance
• Moving average indicators
• Buy/sell signal generation
• Backtesting engine
• Performance metrics (Sharpe Ratio, Returns)
